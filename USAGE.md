# 快速使用指南

## 最简单的启动方式

### Windows用户
双击运行 `start.bat` 文件，脚本会自动：
1. 创建虚拟环境（如果不存在）
2. 安装所有依赖
3. 启动应用程序

### Linux/Mac用户
在终端中运行：
```bash
./start.sh
```

## 手动启动步骤

如果自动脚本不工作，可以手动执行以下步骤：

### 1. 创建虚拟环境

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**Linux/Mac:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

**注意**: 首次安装时，会下载所有依赖包，请确保网络连接正常。

### 3. 运行程序

```bash
python run.py
```

**首次运行**: Skyfield会自动下载NASA JPL DE421星历表文件（约17MB），这是正常现象。下载完成后，程序会自动启动。

### 4. 访问应用

打开浏览器，访问：
```
http://localhost:5000
```

## 使用功能

### 选择城市
在页面顶部的下拉菜单中选择您所在的城市或感兴趣的城市。

### 查看月相日历
- 使用"上个月"/"下个月"按钮浏览不同月份
- 或直接在年份和月份选择器中选择

### 查看详细信息
点击日历中的任意日期，即可查看该日期的详细月相信息，包括：
- 月相名称和图标
- 光照比例
- 月升月落时间
- 农历日期
- 天文知识解释

## 停止程序

在运行程序的终端窗口中按 `Ctrl+C` 即可停止服务器。

## 常见问题

### Q: 程序启动后浏览器显示"无法访问此网站"？
A: 请确保：
1. 程序已成功启动（终端显示"Running on http://0.0.0.0:5000"）
2. 浏览器访问的地址是 `http://localhost:5000` 或 `http://127.0.0.1:5000`
3. 端口5000没有被其他程序占用

### Q: pip安装依赖时报错？
A: 请确保：
1. Python版本 >= 3.7
2. pip已升级到最新版本：`pip install --upgrade pip`
3. 网络连接正常

### Q: 首次运行时卡在"下载星历表"？
A: 这是正常现象，Skyfield需要下载约17MB的星历表文件。请耐心等待，通常需要1-3分钟（取决于网速）。

## 系统要求

- **Python**: 3.7或更高版本
- **操作系统**: Windows、Linux、macOS
- **内存**: 最低512MB可用内存
- **磁盘空间**: 约100MB（包括虚拟环境和星历表文件）
- **网络**: 首次运行时需要联网下载依赖和星历表

## 技术支持

如遇到问题，请检查：
1. Python版本是否符合要求：`python --version`
2. 依赖是否正确安装：`pip list`
3. 终端是否有错误信息
